# AbbyCooks

A premium culinary recipe application built with React and Vite.

## Features
- **Sophisticated Design**: Clean, editorial aesthetic with a warm, inviting color palette.
- **Responsive Layout**: Optimized for both desktop and mobile viewing.
- **Recipe Grid**: Beautifully presented recipe cards with meta-information.
- **Modern Tech Stack**: React, Vite, and Vanilla CSS for maximum performance and control.

## Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```
